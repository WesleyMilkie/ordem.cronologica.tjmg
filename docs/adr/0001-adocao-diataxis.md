# ADR 0001: Adoção de Diátaxis

- Status: Aceito
- Data: 2026-03-04

## Contexto

A documentação estava dispersa e sem separação de objetivos, dificultando manutenção e onboarding.

## Decisão

Adotar estrutura Diátaxis com pastas:
- `docs/tutorials`
- `docs/how-to`
- `docs/reference`
- `docs/explanations`
- `docs/adr`

## Consequências

- Facilita encontrar informação por necessidade do leitor.
- Evita mistura de guia de execução com material conceitual.
- Exige disciplina de manter cada documento no tipo correto.
